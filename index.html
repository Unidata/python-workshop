Unidata’s Python Training Workshop

This workshop teaches the use of python in the atmospheric and oceanic sciences, with a particular emphasis on using Unidata’s other technologies through Python.

Getting started

See the installation instructions for information on how to set up a Python environment with the appropriate packages. You can also get a temporary environment from mybinder.org which will let you test out the Python notebooks without downloading anything to your local system.

You can also just view the notebooks on NBViewer to get a feel for what’s in the workshop.
